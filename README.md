# Dashboard Analisis Twitch Streamers

## Deskripsi Proyek

Dashboard ini dibuat untuk melakukan analisis data terhadap dataset Twitch Streamers serta menampilkan hasil Machine Learning menggunakan Streamlit.

## Dataset

Dataset yang digunakan:

https://raw.githubusercontent.com/DachsteinSilalahi/Dataset_kelompok2/main/twitchdata-update.csv

## Fitur Dashboard

* Home Dashboard
* Dataset Overview
* Exploratory Data Analysis (EDA)
* Visualisasi Interaktif
* Analisis Korelasi
* Machine Learning
* Kesimpulan

## Tools dan Library

* Python
* Streamlit
* Pandas
* Plotly
* Scikit-Learn

## Cara Menjalankan

Install dependency:

```bash
pip install -r requirements.txt
```

Jalankan aplikasi:

```bash
streamlit run app.py
```
